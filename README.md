# ServisPro

ServisPro je desktop aplikacija za evidenciju servisnih zahtjeva, klijenata, uređaja, popravaka, statusa i izvještaja.

Projekt se izrađuje za kolegij Napredne tehnike programiranja.

## Tehnologije

- C++ Builder / RAD Studio
- VCL Windows aplikacija
- Baza podataka
- GitHub za timski rad

## Planirane funkcionalnosti

- prijava korisnika
- evidencija klijenata
- evidencija uređaja
- evidencija servisnih zahtjeva
- pretraga, filtriranje i sortiranje
- spremanje postavki aplikacije
- generiranje izvještaja
- kriptiranje osjetljivih podataka
- REST ili SOAP integracija
- statička ili dinamička biblioteka

## Struktura projekta

- `app/` - izvorni kod aplikacije
- `database/` - baza podataka i SQL skripte
- `docs/` - dokumentacija i prijavnica
- `exports/` - generirani izvještaji
- `assets/` - slike, ikone i ostali resursi
